## A replacement for switch statements in Javascript
In Javascript, one can replace a switch statement using object literal lookups like so:

```
  return {
  0 : () => setDefaultAddress(),
  1 : () => editAddress(),
  2 : () => deleteAddress(),
  }[index]();
```
Instead of:


```
switch (index) {
  case 1:
    setDefaultAddress()  
  break;

  case 2:
    editAddress()      
  break;

  case 3:
    deleteAddress()    
  break;
}
```
